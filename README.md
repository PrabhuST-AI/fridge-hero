# Fridge Hero 🧑‍🍳

Fridge Hero is a modern, AI-powered recipe generator that helps you turn whatever you have in your fridge into culinary masterpieces. Simply enter three ingredients, and let the magic of Groq AI suggest a delicious recipe for you.

![Fridge Hero Screenshot](src/assets/hero.png)

## ✨ Features

- **AI-Powered Recipes**: Utilizes the high-speed Groq API to generate creative recipes.
- **Minimalist Interface**: Simple, 3-ingredient input system for quick results.
- **Premium Design**: Built with a sleek, modern aesthetic featuring:
  - Deep glassmorphism effects.
  - Vibrant purple gradients.
  - Responsive layout for all devices.
  - Custom typography using the "Outfit" font.
- **Secure API Integration**: Direct API key input for privacy and flexibility.

## 🚀 Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) installed on your machine.
- A [Groq API Key](https://console.groq.com/keys).

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/PrabhuST-AI/fridge-hero.git
   cd fridge-hero
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Start the development server:**
   ```bash
   npm run dev
   ```

4. **Open your browser:**
   Navigate to `http://localhost:5173` (or the port shown in your terminal).

## 🛠️ Built With

- **Vite** - Next Generation Frontend Tooling
- **Groq SDK** - High-speed AI Inference
- **Vanilla JavaScript** - Core logic and DOM manipulation
- **CSS3** - Modern styling with Flexbox/Grid and Transitions

## 📖 How to Use

1. Enter your **Groq API Key** in the top input field.
2. Type in **three ingredients** you have on hand.
3. Click the **"Cook Magic"** button.
4. Wait a few seconds for the AI to whisk up a recipe!
